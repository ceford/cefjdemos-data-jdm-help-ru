## Описание

Тип пункта меню **Форма входа** используется для создания страницы, содержащей форму входа.

## Как получить доступ

Чтобы создать новый элемент меню **Форма входа**:

- Выберите **Меню → \[название меню\]** в меню администратора
  (например, **Меню → Главное меню**). Затем...
  - Нажмите кнопку «Новый» в панели инструментов. Затем...
  - Нажмите кнопку выбора типа элемента меню.
  - В модальном диалоге выберите пункт «Пользователи», чтобы открыть список, а затем
    выберите элемент **Форма входа**.

Чтобы отредактировать существующий элемент меню Форма входа:

- Выберите его заголовок в списке *Меню: Элементы*.

### Вкладка «Опции»

#### Панель входа

- **Выберите тип перенаправления после входа** Это может быть *Пункт меню* или *Внутренний URL*. Следующие поля изменяются в зависимости от настройки этого параметра. Для многоязычного сайта рекомендуется использовать *Пункт меню*.
  - **Перенаправление после входа по пункту меню** Выберите или создайте пункт меню для страницы, на которую будет выполнено перенаправление после успешного входа. Если пункт меню не выбран, пользователи будут перенаправлены на свой профиль после входа.
  - **Перенаправление после входа** Выберите внутренний URL для перенаправления после входа.
- **Описание входа** Показать или скрыть описание входа.
- **Текст описания входа** Введите текст, который нужно отобразить на странице входа.
- **Изображение для входа** Выберите или загрузите изображение для отображения на странице входа.
- **Описание изображения (Альтернативный текст)** Необходимо для экранных читалок для обеспечения доступности.
- **Без описания** Чекбокс для выбора, если изображение является исключительно декоративным и не требует объяснений.

#### Панель выхода

Эта панель использует те же заголовки полей для управления процессом выхода.

## Советы

- **URL для входа и выхода** может быть использован, чтобы отправить пользователя на определенную страницу, созданную для предоставления обратной связи пользователю. Например, страницу с названием *Вы теперь вошли в систему* с некоторой общей информацией. Использование модуля пользователя для отображения ссылок на обновление профиля пользователя, просмотр профиля пользователя и других функций пользователя может быть показано на этой странице, улучшая опыт использования сайта.
