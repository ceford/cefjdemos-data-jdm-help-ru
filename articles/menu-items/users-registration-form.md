## Описание

Тип элемента меню *Форма регистрации* используется для создания страницы с формой регистрации пользователя. Форма по умолчанию содержит основную информацию: Имя, Имя пользователя, Пароль и Адрес электронной почты.

## Как получить доступ

Чтобы создать новый пункт меню **Форма регистрации**:

- Выберите **Меню → \[название меню\]** из меню администратора
  (например, **Меню → Главное меню**). Затем...
  - Выберите кнопку "Новый" на панели инструментов. Затем...
  - Выберите кнопку выбора типа пункта меню.
  - В модальном диалоге выберите пункт "Пользователи", чтобы открыть список, и затем
    выберите пункт **Форма регистрации**.

Чтобы редактировать существующий пункт меню Форма регистрации:

- Выберите его заголовок в списке *Меню: Элементы*.
