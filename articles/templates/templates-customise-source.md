## Описание

Страница *Templates: Customise Source* — это место, где редактируется исходный код файлов шаблона. Она предоставляет текстовый интерфейс для редактирования файлов шаблона. Синтаксис программирования HTML и PHP подсвечивается, чтобы упростить чтение исходных файлов кода. В строке заголовка слово *Source* появляется как название шаблона, например *(Cassiopeia)*.  

## Как получить доступ

- Выберите **Система → Панель шаблонов → Шаблоны сайта** в меню администратора. Или...
- Выберите **Система → Панель шаблонов → Шаблоны администратора** в меню администратора. Затем...
  - Выберите имя шаблона в колонке **Шаблоны**. Затем...
    - Выберите файл для редактирования на вкладке Редактора.

## Поля формы

### Вкладка Редактора

- Выберите файл для редактирования. Область редактирования отображает текст с подсветкой синтаксиса для большинства типов файлов.

### Вкладка Создания Переопределений

- Выберите элемент для переопределения. Если выбрана папка, она разворачивается, чтобы показать список файлов. Если выбран файл, он сразу копируется в папку html без запроса на подтверждение. Переопределение помещается в соответствующее место. Выдается сообщение о подтверждении, например: *Переопределение создано в /templates/cassiopeia/html/mod_whosonline*.

### Вкладка Обновленных Файлов

Если с момента создания переопределений в шаблон не вносились обновления, эта вкладка будет содержать простое сообщение:

- **Уведомление** Переопределенные файлы актуальны. Ничего не было изменено в последнем обновлении расширения или Joomla.

Если были обновления, в таблице будет показан список переопределений, которые необходимо перепроверить.

### Вкладка Описания Шаблона

- **Эскиз и Описание** Информация об этом шаблоне.

## Панель инструментов

Иконки на панели инструментов меняются в зависимости от выполняемого действия. Вы можете увидеть:

- **Сохранить** Сохраняет объект и остается на текущем экране.
- **Сохранить и закрыть** Сохраняет объект и закрывает текущий экран.
- **Копировать шаблон** Копирует текущий шаблон. Откроется диалоговое окно с просьбой ввести новое имя.
- **Предпросмотр шаблона** Выберите, чтобы открыть сайт в новой вкладке браузера.
- **Управление папками** Позволяет создавать и удалять папки шаблонов с помощью диалогового окна.
- **Новый файл** Позволяет загрузить файл с вашего компьютера в иерархию файлов шаблона с помощью диалогового окна.
- **Переименовать файл** Выберите файл для редактирования. Нажмите кнопку "Переименовать", чтобы ввести новое имя.
- **Удалить файл** Вам будет предложено подтвердить или отменить действие.
- **Закрыть файл** Закрывает открытый файл и возвращается на вкладку редактора.
- **Помощь** Открывает этот экран помощи.

## Советы

- Важно: не удаляйте шаблонные файлы по умолчанию через FTP, так как
  это приводит к ошибкам как на стороне клиента, так и на стороне сервера.
- Перед редактированием HTML и CSS файла шаблона, желательно сделать резервную копию файла, который вы редактируете. Также, вы можете редактировать эти файлы вне Joomla! с помощью HTML или CSS редактора на ваш выбор.
